# 创建云企业外接程序
通过使用 Visual Studio 中的"云企业外接程序"模板，您可以快速、轻松地创建托管在 SharePoint for Office 365 或 SharePoint 2013 中且基于 Visual Studio LightSwitch 技术的业务线外接程序。用户可以在台式计算机和大多数移动设备上下载和运行云企业外接程序。
