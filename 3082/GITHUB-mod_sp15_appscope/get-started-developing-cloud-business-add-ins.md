# Introducción al desarrollo de complementos empresariales de nube
La mejor manera de aprender a desarrollar un complemento empresarial de nube para SharePoint es comenzar y probar. Los siguientes tutoriales y ejemplos de asociados le guían a través del proceso paso a paso. Anímese; es mucho más fácil de lo que podría imaginar.

  
    
    


## Tutorial de administrador de incidentes

En este tutorial, creará un complemento para que un departamento de bomberos ficticio de Contoso se encargue de la administración de incidentes in situ. El tutorial cubre los conceptos fundamentales para crear un complemento empresarial de nube y también conceptos más avanzados como la integración de bibliotecas de documentos y listas de SharePoint. Consulte  [Administrador de incidentes: Tutorial de complemento empresarial de nube](incident-manager-a-cloud-business-add-in-tutorial.md) para ver el tema del tutorial y [Administrador de incidentes: Un tutorial de complemento empresarial de nube](http://code.msdn.microsoft.com/Incident-Manager-A-Cloud-c32d9b04) para ver el ejemplo.
  
    
    

## Tutorial de rastreador de candidatos

En este tutorial, creará un complemento para realizar un seguimiento de los candidatos a un puesto, aprovechando la característica de integración social de SharePoint para Office 365 con el fin de proporcionar un suministro de noticias para el complemento. Consulte  [Procedimiento para crear una aplicación empresarial de nube con un suministro de noticias de redes sociales](create-a-cloud-business-add-in-with-a-social-newsfeed.md) para ver el tema del tutorial y [Creación de un complemento empresarial de nube con un suministro de noticias sociales](http://code.msdn.microsoft.com/Creating-a-Cloud-Business-8540c0c9) para ver el ejemplo.
