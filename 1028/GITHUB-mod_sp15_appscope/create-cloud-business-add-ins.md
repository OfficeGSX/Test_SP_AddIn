# 建立雲端商務增益集
使用 Visual Studio 中的雲端商務增益集範本，您可以快速和輕鬆建立企業營運系統增益集是架設 SharePoint 上的 Office 365 或 SharePoint 2013 和 Visual Studio LightSwitch 技術為基礎。使用者可以下載及桌上型電腦和最行動裝置上執行雲端商務增益集。
