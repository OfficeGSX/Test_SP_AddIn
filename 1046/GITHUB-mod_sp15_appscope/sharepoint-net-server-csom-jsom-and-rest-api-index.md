# Índice de servidor do SharePoint 2013 .NET, CSOM, JSOM e API REST

## Índice da API do SharePoint

Use o índice de API procurar muitos dos tipos de usadas com mais frequência e objetos que são implementados no modelo de objeto do servidor .NET e pelo menos um modelo de programação de cliente: modelo de objeto do cliente .NET (CSOM), o modelo de objeto JavaScript JSOM () e/ou o restante.
  
    
    
A tabela 1 lista o núcleo mais frequentemente usado, APIs, que são na maioria dos casos, com base em tipos de implementação do servidor .NET. Em alguns casos, tipos são nativos do SharePoint de programação do cliente e não há nenhum tipo de servidor equivalente do .NET. Em outros casos, algumas, mas nem todas as possíveis cliente programação modelo implementações de um tipo específico existem.
